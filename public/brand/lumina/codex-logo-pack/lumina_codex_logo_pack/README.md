# Lumina Logo / Brand Asset Pack

This pack is for the Midnight Sapphire / Porcelain / Brushed Gold Lumina direction.

## Suggested usage

- Header / navbar: `transparent/lumina-primary-wordmark-gold-transparent.png`
- Dark UI wordmark: `transparent/lumina-wordmark-gold-transparent.png`
- Light UI wordmark: `transparent/lumina-primary-wordmark-navy-gold-transparent.png`
- App launcher / PWA icon: `app-icons/lumina-app-icon-dark-1024.png`
- Favicon: use the files in `favicons/`
- Small UI/avatar mark: `transparent/lumina-small-icon-gold-transparent-512.png`
- Reference image for Codex: `brand-reference/lumina-selected-third-brand-image.png`

## Notes

These assets are PNG exports designed to be implementation-friendly. The app icons are intentionally opaque because PWA/mobile icons usually render better with a fixed background. The transparent assets are best for navbars, empty states, loading screens, and dark UI surfaces.

Keep the logo clear space generous, avoid placing it over busy artwork, and prefer Midnight Sapphire / Deep Navy backgrounds for the gold version.
